Summary

Agricultural package diffusion incomplete: not all components spread to all regions. Egypt too warm for einkorn wheat. Components arrived at different times (sheep before cereals in southwestern Europe). Some regions domesticated limited local crops (poppies, watermelons). Most outlying agriculture came via diffusion, not independent domestication. Geographic constraints modified package content.

Key Concepts

Diffusion adaptability: packages modified locally based on climate/geography. Temperature constraints prevent crop adoption (einkorn unsuited Egypt heat). Timing variation: animals/crops arrived separately. Local domestication: limited to climate-suited wild species. Geographic determinism applies to diffusion: climate determines which package elements viable.

Historical & Geographic Context

Fertile Crescent package spread but modified: Egypt received sheep first then cereals, substituting locally-adapted crops. Western Europe adapted sheep pastoralism before grain farming. Local domestications: poppies (Europe), watermelons (Egypt) supplemented diffused crops. Most regions received diffused agriculture rather than independently inventing.

Connection to Main Thesis

Geographic climate determines package modification during diffusion. Regions adapted diffused package to local conditions. Geographic constraints on crops maintained through diffusion. Egypt's climate unsuitable for einkorn but suitable for other cereals. Geographic determinism extends to package modification during technology diffusion.
