Summary

Writing system diffusion traceable through letter form comparison. Linear B syllabary (Mycenaean Greece ~1400 BCE) adapted from Linear A (Minoan Crete). Letter forms reveal derivation patterns. Hundreds of writing systems adapted from existing ones. Blueprint copying and modification standard pattern.

Key Concepts

Writing system continuity. Letter form inheritance traces diffusion patterns. Syllabary modification reflects language phoneme differences. Geographic contact enables writing system adoption. Alphabet/syllabary adaptation easier than independent invention.

Historical & Geographic Context

Minoan Linear A: independent invention or Egyptian influence (unclear). Mycenaean Greeks: geographic contact with Minoans enabled Linear A acquisition. Linear B adaptation: Modified Linear A for Greek phonetic differences. Geographic proximity determined writing system transmission.
